# MzansiBuilds-DerivcoCodeSkillsQuest
A platform that helps developers to build publicly and keep up with what other developers are building.

## Setup & Installation
1. Clone the repository:
   `https://github.com/Its-JustSbu/MzansiBuilds-DerivcoCodeSkillsQuest.git`
2. Navigate to Backend and Follow Backend README.md instructions:
   `cd Backend`
3. Navigate to Frontend and Follow Frontend README.md instructions:
   `cd frontend`
